# SSH Tunneling Module

Labs and exercises covering SSH tunneling techniques, port forwarding, and multi-hop pivoting.

## Labs

| # | Lab | Topic |
|---|-----|-------|
| 01 | SSH Basics | SSH key auth, config, and connection fundamentals |
| 02 | Local Forwarding | Forward local ports to remote services |
| 03 | Remote Forwarding | Expose local services to remote networks |
| 04 | Dynamic Forwarding | SOCKS proxy tunneling with SSH |
| 05 | Multi-Hop Chaining | Chain SSH tunnels across multiple hosts |
| 06 | Double Pivot | Two-layer pivot through segmented networks |
| 07 | ICMP Tunnel | Tunneling traffic over ICMP |
